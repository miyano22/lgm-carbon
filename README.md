# LGM Ocean Carbon Cycle Analysis

This repository contains analysis and visualization scripts used in
Nishida et al. (submitted to AGU).

The scripts reproduce the figures shown in the manuscript using
processed model outputs.

## Contents
- Python and GMT scripts for figure generation

## Data availability
Model output data are archived separately.
PMIP forcing data are publicly available from the PMIP database.
